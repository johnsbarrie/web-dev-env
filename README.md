# Build example for Gamesys 

> A gulp and webpack example for client and server

> Install client

```
cd client
npm install
bower install
```

> Run javascript dev for client ([localhost:8080](http://localhost:8080)).

```
cd client
gulp webpack:dev
```

> Run javascript dev for css ([localhost:8080](http://localhost:8080)).

```
cd client
gulp webpack:dev
```
